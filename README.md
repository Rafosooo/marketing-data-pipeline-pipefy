# Marketing Data Pipeline

This project extracts marketing data from Pipefy API and loads it into a PostgreSQL database for BI analysis.

## Stack
- Python
- REST API
- PostgreSQL
- Power BI

## Pipeline

Pipefy API → Python ETL → PostgreSQL → Power BI Dashboard

## Features
- Automated data extraction
- Scheduled execution
- Data cleaning and normalization
- Ready-to-query tables for BI tools

## Business Impact
Reduced manual reporting time from 6h/week to fully automated dashboards.
